# KCSA practice exam app

The app is available on GitHub Pages:

https://serquicky.github.io/kcsa-practice-exam/

An interactive, browser-based KCSA practice exam app with two modes:
- **Exam Mode** (timed, scored at the end)
- **Practice Mode** (instant feedback with explanations)

Feel free to add or correct questions through a PR.
